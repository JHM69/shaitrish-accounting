# Shaitrish Company Tracker

Financial ledger and operational dashboard for Shaitrish (shaitrish.com): expense ledger, client invoices & MRR, founder equity, double-entry journal, monthly USD→BDT exchange rates, and Bangladesh (NBR) tax compliance notes.

Data persists in the browser's `localStorage` — use the in-app export/import backup controls before clearing browser data or switching machines.

## Stack

React 19 + TypeScript + Vite + Tailwind CSS 4 + Recharts.

## Run locally

```sh
npm install
npm run dev      # http://localhost:3000
```

## Other scripts

```sh
npm run build    # production build to dist/
npm run preview  # serve the production build
npm run lint     # typecheck (tsc --noEmit)
```
